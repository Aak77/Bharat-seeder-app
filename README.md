# Bharat Seeder 🌾

**Agricultural Logistics & Seeding Management Application**

A React Native mobile application designed to streamline agricultural seeding operations with dual-mode functionality for Farmers and Operators.

## 🚀 Features

- **Custom Animated Splash:** High-fidelity fade-in and scale animation for the Smart Seed logo.
- **Authentication:** Secure mobile number and OTP-based login flow.
- **Dual-Role Dashboard:** Custom interfaces for 'Farmer' and 'Operator' roles.
- **Modern UI:** Built using React Native Paper and specialized agricultural design patterns.

## 🛠️ Tech Stack

- **Framework:** React Native (Expo)
- **Navigation:** React Navigation (Stack)
- **Language:** JavaScript
- **Tools:** Cursor/VS Code, GitHub

## 🏃‍♂️ How to Run

1. Clone the repo: `git clone https://github.com/Aak77/Bharat-seeder-app.git`
2. Install dependencies: `npm install`
3. Start the project: `npx expo start`
